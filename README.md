Sales Analysis Dashboard (Power BI)
​Bu layihədə satış məlumatlarının vizuallaşdırılması və əsas performans göstəricilərinin (KPI) analizi üçün interaktiv Power BI dashboard-u hazırlamışam.
​📊 Əsas Göstəricilər və Analizlər:
​KPI Metrikləri: Ümumi satış həcmi (227.6M), sifariş sayı və satış trendləri.
​Satış Dinamikası: İllərə və aylara görə satış vahidlərinin artım/azalma analizi (Waterfall chart).
​Sifariş Statusu: Bağlanmış, açıq və digər statuslar üzrə sifarişlərin paylanması (Funnel & Donut chart).
​Məhsul Analizi: Aylıq alış qiymətləri və satış həcmi üzrə müqayisəli vizuallar.
​🛠️ İstifadə olunan Texnologiyalar:
​Power BI: Data modelləşdirmə və vizuallaşdırma.
​DAX: Dinamik hesablamalar və ölçülərin (measures) yaradılması.
​Power Query: Məlumatların təmizlənməsi və transformasiyası (ETL).
